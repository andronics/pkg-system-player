# Media Player Control - MPV/media playback management

**Package**: `@system/player`
**Version**: 0.1.0
**Repository**: `pkg-system-player`

## Overview

Media Player Control - MPV/media playback management

## Installation

### Prerequisites

No external dependencies required.

### Using pkg-cli

```bash
# Install from GitHub
pkg-cli install @system/player

# Or install from local source
cd ~/.pkgs
stow player
```

## Usage

### System Integration

This package provides system integration for media player control - mpv/media playback management.


## Configuration

Configuration files are typically stored in:
- `~/.config/player/` - User configuration
- `~/.local/share/player/` - Application data

## Uninstallation

```bash
# Using pkg-cli
pkg-cli uninstall @system/player

# Or manual unstow
cd ~/.pkgs
stow -D player
```
